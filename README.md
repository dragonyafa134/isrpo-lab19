## Описание

В ходе данной лабораторной работы было проведено знакомство с основами языка программирования JavaScript. Изучена его роль в веб-разработке, а также выполнено сравнение базовых концепций JavaScript и C#. 

### В рамках работы было изучено:

1. **Подключение JavaScript к HTML** — использование тега `<script>`.
2. **Циклы**: `for`, `while`, `do...while`, операторы `break` и `continue`, вложенные циклы.
3. **Функции**: классическое объявление, параметры по умолчанию, функции без `return`, функции как значения, стрелочные функции.
4. **Массивы**: создание, основные методы (`push`, `pop`, `shift`, `unshift`, `splice`, `slice`, `concat`, `indexOf`, `includes`), перебор (`for...of`).
5. **Объекты**: создание через литерал, доступ к свойствам, добавление и удаление свойств, методы объектов, перебор свойств (`for...in`), вложенные структуры.
